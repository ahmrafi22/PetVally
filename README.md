# PetVally - Next.JS 


## About This Project

![Petvally banner](./public/petvally.png)

### Key Modules & Features

**Module 1: Shopping & Pet Adoption**
- Storefront for pet food, medicine, toys, and pets
- Lifestyle-based pet recommendations
- Shopping cart, checkout, order tracking
- Product reviews and ratings
- Search and filter pets for adoption

**Module 2: Pet Donations & Missing Pets**
- Post pets for donation and adoption forms
- Report and update missing pets
- Location-based lost pet alerts and notifications

**Module 3: Veterinary Services & Admin Controls**
- AI chatbot for veterinary advice (text & image)
- Vet directory and appointment scheduling
- Admin dashboard for inventory and order management

**Module 4: Pet Care Jobs & Companion Services**
- Post and apply for pet care jobs (boarding, sitting, walking)
- Pet companion profiles and calendar view
- Job application management and earnings tracking
